# 小西的技术笔记

> 分享前端开发经验、实用工具推荐、技术学习心得

## 📖 博客简介

这是一个基于 Hexo 搭建的技术博客，分享前端开发、JavaScript/TypeScript、React 等技术文章。

## 🛠 技术栈

- **博客框架**：Hexo 7.x
- **主题**：Landscape
- **部署**：GitHub Pages
- **构建工具**：Node.js 22.x
- **写作语言**：Markdown

## 🚀 快速开始

### 本地预览

```bash
# 安装依赖
npm install

# 启动本地服务器
hexo server

# 访问 http://localhost:4000/blog/
```

### 写文章

```bash
# 创建新文章
hexo new post "文章标题"

# 生成静态文件
hexo generate

# 部署到 GitHub
hexo deploy
```

## 📂 目录结构

```
blog/
├── source/          # 源文件（文章、页面）
│   └── _posts/      # 博客文章
├── themes/          # 主题文件
├── public/          # 生成的静态文件
├── _config.yml      # 配置文件
└── package.json     # 依赖配置
```
