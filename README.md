# 小西的技术笔记

> 分享前端开发经验、实用工具推荐、技术学习心得

## 技术栈

- Hexo 7.x
- Node.js 22.x
- GitHub Pages
- Markdown

## 快速开始

```bash
# 安装依赖
npm install

# 本地预览
hexo server

# 写文章
hexo new post "文章标题"

# 部署
hexo generate && hexo deploy
```
